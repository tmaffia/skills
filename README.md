# Personal Agent Skills

A collection of custom Claude Code skills for personal automation and workflows.

## Available Skills

### tech-assessment
Generate weekly tech assessment reports for mentor/mentee relationships tracking growth, goals, and action items over time. Produces an updated overview.md, a dated assessment.md archival file, and a Microsoft Teams message draft.

## Installation

```bash
# Install tech-assessment skill
npx skills add https://github.com/tmaffia/skills --skill tech-assessment

# Install globally (available across all projects)
npx skills add https://github.com/tmaffia/skills --skill tech-assessment -g
```
